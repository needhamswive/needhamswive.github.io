|:--------------:|:-------:|
| 9/3   -- 9/16  | forward |
| 9/17  -- 9/23  |  back   |
| 9/24  -- 9/30  | inward  |
| 10/1  -- 10/7  |  twist  |
| 10/8  -- 10/14 | reverse |
| 10/15 -- 10/21 | forward |
| 10/22 -- 10/28 |  back   |
| 10/29 -- 11/4  | inward  |
